## Evoxt Speed Test：実際の測定結果はどうだったか

「speed test」というワードを調べていると、大きく 3 つの観点が気になるはずです。

1. **CPU 速度**（シングルコア・マルチコア）
2. **ディスク I/O 速度**
3. **ネットワーク転送速度**

それぞれ独立したサイト VPSBenchmarks が実施したテストデータをベースに見ていきましょう。

---

### CPU パフォーマンス

Evoxt の最大の売りはここです。

VPSBenchmarks が VM-1（1 コア / 2 GB RAM / $5.99/月）プランでテストを実施したところ、**シングルコア性能で他の格安 VPS プロバイダーを大きく上回る**結果が確認されています。Sysbench・Geekbench・エンデュランステスト（24 時間連続稼働）の複合スコアで算出される Raw CPU Capacity グレードも高評価でした。

エンデュランステストで特に注目すべきなのは「**スロットリングがかかるかどうか**」です。一部のプロバイダーは長時間負荷をかけると CPU を制限してきますが、Evoxt のテスト結果では **24 時間通じてパフォーマンスが安定**しており、変動係数（coefficient of variation）も低く抑えられていました。つまり「一瞬だけ速い」ではなく「ずっと速い」という話です。

VPSBenchmarks の総合ランキングでは：

- **2022年** ：$60以下カテゴリで3位
- **2023年** ：$8以下カテゴリで2位、$60以下カテゴリで3位
- **2024年** ：$25以下カテゴリで3位
- **2025年** ：$25以下カテゴリで2位

これだけ長期間にわたって上位をキープしているプロバイダーはそう多くありません。

### ネットワーク速度

ネットワーク転送テストでは、VPS から周辺サーバーへの HTTP ダウンロード・アップロード速度を 4 回計測し、最速値を採用するという方法が使われています（イントラデータセンター間の測定は除外されるため、より実態に近い数値になります）。

**1 Gbps のネットワークポートを全プランに標準装備**している点は評価ポイントです。帯域制限や「バースト速度」のような曖昧な表記はなく、透明な価格設定と一致しています。

拠点によってネットワーク品質は異なります：

- **スタンダード拠点**（米・英・加・独・蘭・波・日（東京）・馬・豪）：主要 Tier 1 ISP との接続
- **プレミアムネットワーク**（香港・日本大阪）：香港は CN2 経由で中国向けルーティングが最適化
- **プレミアムプラス**（マレーシア）：MyIX・Google・Cloudflare とのピアリング

アジア向けトラフィックを扱うなら、香港や大阪のプレミアムプランが効いてきます。ユーザーレビューでも「アジアへのレイテンシが低い」という声が複数見られました。

---

### ディスク I/O

Fio・Sysbench によるディスク性能テストでは、ランダム・シーケンシャル双方の読み書き速度が計測されています。KVM ハイパーバイザーと SSD ストレージの組み合わせで、**データベース集中型のウェブアプリでもレスポンスが安定している**との評価があります。

あるユーザーのコメントを意訳すると「1 コアだけなのにウェブサイトが速い。DBクエリも早い」——これは CPU クロックの高さがそのままディスク I/O の体感速度にも影響している例です。

VM-1 の場合、注文から接続可能になるまで **約 2 分 20 秒**というテスト結果が出ています（VM-4 では 1 分 20 秒、VM-8 では 2 分 50 秒というデータも）。デプロイ速度は速く、待たされる感はほとんどありません。
